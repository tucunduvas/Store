

# 📊 Análise de Vendas - Challenge Data Science (Alura)

Este projeto foi desenvolvido como parte do **Challenge de Data Science da Alura**, com o objetivo de analisar dados de vendas de quatro lojas e extrair insights que possam apoiar a tomada de decisão.

A análise foi realizada utilizando **Python e a biblioteca Pandas**, explorando métricas como faturamento, categorias de produtos, avaliações de clientes e custos de frete.

---

# 🎯 Objetivo do Projeto

O objetivo deste projeto é realizar uma análise exploratória dos dados de vendas de quatro lojas diferentes para identificar:

* Qual loja possui maior faturamento
* Quais categorias de produtos vendem mais
* Como está a avaliação média dos clientes
* Quais produtos têm maior e menor volume de vendas
* Qual loja possui o frete médio mais alto

Essas análises ajudam a entender melhor o desempenho das lojas e identificar oportunidades de melhoria.

# 📈 Análises Realizadas

## 1️⃣ Análise do Faturamento

Foi calculado o faturamento total de cada loja somando o valor de todos os produtos vendidos.

Essa análise permite identificar **qual loja gera maior receita**.

---

## 2️⃣ Vendas por Categoria

Foram analisadas as vendas por **categoria de produto**, permitindo identificar quais tipos de produtos possuem maior demanda.

Essa informação é útil para entender o comportamento de compra dos clientes.

---

## 3️⃣ Média de Avaliação das Lojas

Foi calculada a **média das avaliações dos clientes** para cada loja.

Essa métrica ajuda a entender o nível de satisfação dos consumidores com as compras realizadas.

---

## 4️⃣ Produtos Mais e Menos Vendidos

Foram identificados:

* Os **produtos mais vendidos**
* Os **produtos com menor volume de vendas**

Essa análise pode ajudar a identificar produtos de alto desempenho e aqueles que podem precisar de melhorias ou estratégias de marketing.

---

## 5️⃣ Frete Médio por Loja

Foi calculado o **frete médio cobrado por cada loja**, permitindo comparar possíveis diferenças na logística.

O valor do frete pode impactar diretamente a decisão de compra dos clientes.

---

# 📊 Resultados

A análise permitiu comparar o desempenho das quatro lojas considerando diferentes métricas de negócio.

Entre os principais insights obtidos:

* Diferença no faturamento entre as lojas
* Identificação das categorias de produtos mais populares
* Comparação da satisfação dos clientes entre as lojas
* Produtos com maior e menor volume de vendas
* Diferenças no valor médio do frete
